# 文化内容创作智能体 Agent

## 能力总览

| 能力 | 实现方式 | 状态 |
|------|----------|------|
| 文本智能分类与标注 | DeepSeek-chat + 规则双模式 | ✅ |
| 文化要素提取（人物/地点/朝代/技艺/符号） | AI + 规则关键词匹配 | ✅ |
| 文案生成（5平台适配） | DeepSeek-chat | ✅ |
| AI图片生成 | 豆包 Seedream (文生图) | ✅ |
| AI视频生成 | 豆包 Seedance (图生视频) | ✅ |
| 语音合成 | Edge TTS (多音色中文) | ✅ |
| 视频分镜头脚本生成 | DeepSeek-chat | ✅ |
| 内容合规性检查 | 敏感词+历史准确性校验 | ✅ |
| **图片识别理解** | **豆包 doubao-1-5-vision-pro-32k-250115** | ✅ |
| AI+规则双模式 | 无API时可离线运行 | ✅ |

## 多模态架构

```
用户输入
  ├── 纯文字 → DeepSeek-chat (api.deepseek.com)
  └── 带图片 → 豆包 Vision Pro (ark.cn-beijing.volces.com)
       ├── 文物识别：朝代/材质/技法/风格
       ├── 建筑识别：名称/年代/建筑风格
       ├── 非遗识别：项目类型/传承特点
       └── 文旅识别：景点名称/文化背景
```

## 使用方式

### Web平台
启动前后端后，访问 http://localhost:5173
- 导航栏 **"04. 智能体查询"** → 文字问答 + 🖼️图片上传识别
- 导航栏 **"🎨 文案/图片/视频制作"** → AI多模态内容创作

### API调用
```
POST /api/agent/chat       — 智能对话（传 image 字段=base64 触发视觉识别）
POST /api/agent/generate   — 多模态内容生成（文案/图片提示词/视频脚本）
POST /api/agent/classify   — 智能分类标注
POST /api/agent/compliance — 合规检查
POST /api/agent/speech     — 语音合成
POST /api/gen/image        — 豆包Seedream图片生成
POST /api/gen/video        — 豆包Seedance视频生成
```

## 依赖
- DeepSeek API Key (deepseek-chat，纯文本对话)
- 豆包 Volcengine ARK API Key
  - doubao-1-5-vision-pro-32k-250115（图片识别）
  - doubao-seedream-4-5-251128（图片生成）
  - doubao-seedance-1-0-pro-250528（视频生成）
- Edge TTS (免费，无需API Key)
