## 制作指南

### 方案A：屏幕录制（推荐）
1. 运行Streamlit应用：`cd web && streamlit run app.py`
2. 使用OBS Studio或Windows自带录屏（Win+G）录制
3. 按照上述脚本顺序操作演示
4. 后期配音或同步录音
5. 使用剪映/CapCut剪辑，控制在3分钟以内

### 方案B：PPT动画 + 录屏混合
1. 制作简洁的PPT介绍页面
2. 核心功能部分使用真实录屏
3. 使用剪映合成

### 所需素材
- 西安文化主题AI图片（用AI图像生成工具创建）
- 背景音乐：西安鼓乐片段（可使用ace-step技能生成）
- 片头/片尾字幕模板
